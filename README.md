# tattoo-studio
This is the website of my real tattoo studio, located in Santiago de Chile. Today is a static informative website, but in a second stage it incorporates a scheduling system and a reservation payment system, for the management of clients.

## Built With

- HTML, CSS y JS.
- LeadMark Template Created By DevCrud.

## Live Demo

https://gfrancoarq.github.io/tattoo-studio/

## Getting Started

**Instructions to set up this project locally and use as a template:**

- Fork the repo.
- Clone your fork.
- Create a new branch.
- Make your changes, commit what you have made, and make a pull request.
- If you get things working, and it pass all the test, create a pull request.
- Get merged!
